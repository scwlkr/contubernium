# Faber Skill

## Use When

- A task requires backend code, APIs, data models, or server logic

## Checklist

1. Read the invocation objective, constraints, dependencies, and relevant files.
2. Implement only the backend changes required by the assignment.
3. Record concrete artifacts and any interface assumptions.
4. Return the result to `decanus` without redefining scope.

## Completion Signal

The assigned backend work is implemented or a precise blocker has been recorded.
