# Praeco Skill

## Use When

- A task requires release notes, launch copy, or outward-facing messaging

## Checklist

1. Read the shipped behavior and audience context.
2. Write only what is supported by the actual product.
3. Keep the message aligned with the current brand direction.
4. Return the copy result to `decanus`.

## Completion Signal

The requested messaging artifact is complete and tied to verified behavior.
