# Praeco Skill

## Role Summary

Write bounded user-facing messaging tied to verified behavior.

## Capability Domains

- Release notes
- Launch copy
- Product messaging
- Audience adaptation

## Workflow

1. Read the shipped behavior and audience context.
2. Write the requested messaging artifact.
3. Return the copy plus assumptions or blockers.

## Action Selection

- Use `WRITE_MESSAGE` for release notes and outward-facing copy tasks.

## Output Structure

Return a specialist result JSON object with copy artifacts, blockers, and confidence.
