# Explorator Skill

## Use When

- A task requires documentation lookup, project analysis, or option comparison

## Checklist

1. Read the exact research objective and scope limits.
2. Gather the minimum evidence needed to answer it.
3. Summarize findings with tradeoffs and confidence.
4. Return decision-ready research to `decanus`.

## Completion Signal

The research question is answered with specific evidence or a concrete gap is documented.
