# RESEARCH_SCOPE

## Purpose
Gather decision-ready evidence without implementing the solution.

## When to Use
- The assignment is research, documentation lookup, or local project analysis

## Inputs
- Research objective
- Scope limits
- Relevant files or external docs

## Constraints
- Stop at research
- Prefer direct evidence and tradeoffs

## Process
1. Collect the minimum evidence required.
2. Compare options or summarize findings.
3. Return concise decision-ready output.

## Output
A structured specialist result with findings, options, and confidence.

## Failure Modes
- Missing source material: report the gap
- Decision requested: return options, not final authority

## Example
`explorator::RESEARCH_SCOPE` to compare two local model adapters.
